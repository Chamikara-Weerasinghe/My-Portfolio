# Chamikara Weerasinghe - Portfolio

Premium personal portfolio website built with Next.js, React, TypeScript, Tailwind CSS, Framer Motion, and React Three Fiber.

**Live domain:** [chamikara.online](https://chamikara.online)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Environment Variables

Optional — improves GitHub API rate limits:

```env
GITHUB_TOKEN=your_github_personal_access_token
```

## Tech Stack

- Next.js 15 (App Router)
- React 19
- TypeScript
- Tailwind CSS 4
- Framer Motion
- React Three Fiber + Three.js
- GSAP (available)
- Lucide Icons

## Customization

Edit content in `src/lib/data.ts` — profile info, projects, skills, experience, and certificates.

## Deployment

Deploy to Vercel, Netlify, or any platform supporting Next.js. Point your domain `chamikara.online` to the deployment.
